# referencias/ — imagens-protótipo

Base visual do design, **enviada pelo Daniel**. São os protótipos/inspirações que orientam o
projeto do planner de estudos (fotos, scans, prints de concorrentes, mockups, etc.).

## Como catalogar (sugestão)

Ao salvar cada imagem, nomear de forma rastreável e registrar a origem:

```
NN-fonte-descricao.ext        # ex.: 01-etsy-grade-semanal.png
```

E manter um `fontes.md` neste diretório com uma linha por imagem:

| Arquivo | O que é | Origem (URL/quem enviou) | Data |
|---|---|---|---|
| 01-... | grade semanal de estudo | … | 19/jun/2026 |

> Material-base **imutável**: anotações, recortes e análises sobre estas imagens vão para
> `../pesquisa/` e `../conceitos/`, não aqui.
