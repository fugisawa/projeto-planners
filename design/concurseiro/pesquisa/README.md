# pesquisa/ — evidência de design

Resultado da investigação de design feita com **Exa**, **Tavily** e **Playwright** a partir
dos links que o Daniel passar (e das buscas derivadas deles).

```
pesquisa/
├── fontes/      extrações e links datados (Exa/Tavily): texto, specs, preços, achados
└── capturas/    screenshots de navegação (Playwright): telas de lojas/galerias/produtos
```

## Convenções

- **Tudo datado e com fonte:** cada achado registra **de onde veio** e **quando** foi captado
  (datas absolutas). Confiança no padrão do dossiê: `[confirmado]` · `[estimativa triangulada]`
  · `[a confirmar]`.
- **Capturas** nomeadas de forma rastreável, ex.: `NN-dominio-tela.png`
  (`01-etsy-listing-planner.png`), com a URL e a data anotadas num `fontes.md`.
- **Triangular** referências centrais em ≥2 fontes antes de tratá-las como padrão de mercado.
