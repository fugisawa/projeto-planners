# conceitos/ — direção de design

Onde a pesquisa vira **decisão de design**. Saída do trabalho de análise, a partir de
`../referencias/` (protótipos) + `../pesquisa/` (evidência).

## O que mora aqui

- **User stories** — como o concurseiro usa o planner (ex.: *"como concurseiro, quero montar
  meu ciclo de estudo por disciplina e marcar revisões espaçadas para não esquecer o que já vi"*).
- **Features / sistema de páginas** — grade do ciclo, tracker de revisão espaçada, registro de
  desempenho, metas, índice; o que cada página faz e por quê.
- **Direção de design** — moodboard, paleta, tipografia, grid, formato, papel/acabamento.
- **Decisões** — escolhas tomadas, alternativas descartadas e o motivo (rastreável).

> Itens ainda em aberto entram como `[a confirmar]`. Nada aqui é "fechado" sem passar pela
> validação no padrão do dossiê.
